# Midwest Attic

Landing page for Midwest Attic — web design, online presence, and custom tools for small businesses. A DBA of Space Junk Studios, based in Detroit & Southeastern Michigan.

Deployed via Firebase Hosting to **https://midwest-attic.web.app**.

## Stack

Single-file static HTML/CSS. No build step. Deploy with:

```
firebase deploy
```

## Note

This project fully replaced its previous purpose. The original content (a "Let's Make It Legal" landing page with Firestore and Storage config) was removed entirely. The Firebase project ID (`midwest-attic`) and hosting setup were kept; only the public content was swapped.
